VM-LAN: Virtual Network

VM-LAN Exercises

  1. Set up virtual networked clients in Virtualbox
    1. install Windows client Initial setup of a a Windows 7 host, from an .ISO image.
    2. install Linux client Initial setup of a Linux host, from an .ISO image.
    3. install Windows client Initial setup of a a Windows 7 host, from an .OVA appliance.
    4. verify client communications Communicate between Windos and Linux clients, and create a backup copy.
  2. Layer-2 neighbor discovery
    Use Wireshark to observe MS' LLTD and IEEE 803.1ab's LLDP layer-2 protocols in action.
  3. ASSIGNMENT: craft LAN frames with scapy
    Create Ethernet frames and payloads, and observe them in scapy and in Wireshark.
  4. set static IP addresses
    Give the hosts static private IP addresses, verify that they can only communicate with each other.
  5. install access router
    Add VyOS router appliance to the private LAN; it should connect to other students' routers through the physical lab's switch.
  6. ASSIGNMENT: Send IP and TCP packets with scapy
    Create and send Layer-3/Layer-4 packets.
  7. Set up dynamic IP addresses
    1. Calculate subnet values Work out a subnet for DHCP to manage.
    2. Configure DHCP Set up DHCP using the calculated subnet values.
  8. Set up and use an FTP server
    1. install Windows' FTP server
    2. ASSIGNMENT: Examine the FTP protocol
      Upload files with FTP.
  9. add Internet-connected gateway-router
    Add another router that provides a link via VirtualBox's manager to the outside world.
  10. ASSIGNMENT: Demonstrate connection to the Internet
    Perform traceroutes to the Internet, and upload the results to the course website.
  11. add web and FTP servers
    Try out a simple webserver and an FTP server; observe their traffic in Wireshark.
